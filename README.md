# Sistema de Gestão de Ludoteca

Projeto acadêmico desenvolvido em Python para gerenciamento de uma ludoteca, permitindo o cadastro de jogos de tabuleiro e o controle de empréstimos e devoluções.

## Funcionalidades

- Adicionar jogos;
- Remover jogos;
- Listar jogos;
- Buscar jogos por mecânica principal;
- Buscar jogos por número de jogadores;
- Registrar empréstimos;
- Registrar devoluções;
- Listar jogos emprestados.

## Tecnologias utilizadas

- Python
- JSON
- Git e GitHub

## Como executar

1. Clone o repositório;
2. Execute o arquivo principal:

```bash
python main.py
